您好，本作品仿照NBA中文网(china.nba.com)制作主页，素材与资源全部来源于该网站，代码纯手打，绝无抄袭。
本作品使用html5 css3 js jqurey 编写。
作品特点及主要功能简介：
1.头部导航（header）鼠标滚动后固定在页面顶部，且选项卡包含鼠标移入、移出下拉菜单。
2.赛程表（schedule）日期切换按钮根据长度自适应切换，比赛场次根据右侧自动生成，右侧详细场次内容长度左右滑动按钮自适应启用、禁用状态。
3.轮播图（slider）为自适应长度自动滚动播放，左右按钮可实现手动播放，下方圆点可实现跳转。
4.新闻及最新资讯“换一换”按钮自动更换下一组新闻，根据新闻组数自适应，最新资讯图片部分有鼠标移入效果。
5.排名部分（ranking），数据由外部文件管理，页面自动提取内容显示。
6.视频部分三个板块根据自身内容长度，自适应出现左右滚动按钮，并有滚动效果。
7.数据部分赛季数据可自适应出现左右滚动按钮并实现滚动效果。
8.社交媒体（socialMedia）图标鼠标移入移除，显示隐藏二维码效果。
